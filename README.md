# QMK - custom keyboards

![Firmware](https://img.shields.io/badge/firmware-QMK-blue)
![Language](https://img.shields.io/badge/language-C-blue)
![License](https://img.shields.io/badge/license-GPL--2.0-blue)

This repository documents my builds of handwired mechanical keyboards. It contains customized QMK firmware configurations, specific keymaps, and 3D case files. These projects are largely based on open-source designs, which I have adapted to fit my specific needs and avaliable hardware.

## Projects

| Project Name | Type | Key Features | Status |
| :--- | :--- | :--- | :--- |
| [**PlankSS**](keyboards/handwired/plankss) | Ortholinear keyboard, 60% | • 3D printed case and plate<br>• Compact layout<br>• Full Vial GUI support<br>• Audio output switching macro | ✅ Completed |
| [**PScottoFly**](keyboards/handwired/pscottofly) | Split keyboard, 35% | • 3D printed case and plate<br>• Multi-OS Support (Win/Mac)<br>• Macros and tap-dances | ✅ Completed |
| **LeverlesSS** | Arcade controller | • Ergonomic gaming<br>• Hot swappable<br>• Low profile switches<br> | ❇ Concept |
